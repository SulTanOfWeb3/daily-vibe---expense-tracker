# Daily Vibe Check

A simple, clean daily mood tracker to log how you're feeling with quick notes.

## Framework

Vanilla JavaScript with Tailwind CSS (CDN)

## Features

- Track daily mood with 5 emoji options (Amazing, Good, Okay, Rough, Hard)
- Add optional quick notes for each entry
- View total entries and current streak
- All data saved in localStorage
- Delete individual entries or clear all
- Clean, minimal UI with soft shadows and calm color palette

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

Clean Light theme with mood-specific color coding (green for amazing, blue for good, yellow for okay, orange for rough, red for struggling)
